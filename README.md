# [EA-302] OnPhysical-AI
Halla university "OnPhysical AI 하계 몰입과정" 강의자료 
### 강의 목차 
  1일차  
    - STM32 개발환경 구축 및 BareMetal 프로그래밍 개론  
    - STM32 GPIO/HAL 구조 및 디지털 I/O 구현  
  2일차   
    - 디지털 입력·버튼 제어와 디바운싱  
    - ADC 센서 데이터 읽기·필터링·로깅  
  3일차  
    - PWM 출력 제어  
    - Timer 기반 주기 제어와 Non-blocking 구조  
  4일차  
    - 외부 인터럽트(EXTI)와 상태머신 설계  
    - 인터럽트·상태머신 구현 실습  
  5일차  
    - 통합 시스템 설계와 통신·표시장치 개론  
    - RS-485 모터 제어 구현
